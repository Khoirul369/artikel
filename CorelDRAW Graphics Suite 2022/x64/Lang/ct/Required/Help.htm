此為 Setup.exe 中常用命令列的快速參考指南

Setup.exe /Option <必要的參數> [Optional Parameter]

安裝選項：
/a 表示您要建立伺服器影像。 系統管理安裝 - 在網路上安裝產品

Setup.exe /a
/a /q(n) 命令列中的 /q(n) 交換器不再受到支援。請參閱最新版本的《CorelDRAW Graphics Suite 部署指南》，以了解受支援的交換器。

Setup.exe /a /q(n)
/s 用來將 PROFILEID 推送至工作站。PROFILEID 公用屬性可用來將帳戶認證推送至工作站，藉此將所有使用者登入帳戶。

Setup.exe /s PROFILEID=XXXXXXXXXXXXXXX
/x
/uninstall		 表示您要解除安裝軟體。 系統管理安裝 - 在網路上安裝產品

Setup.exe /x


Display Options:
/q[n|b|r|f] 設定安裝的使用者介面之顯示範圍
n - 無 UI
b - 基本 UI
r - 精簡 UI
f - 完整 UI (預設值)

Setup.exe /q
/quiet 安靜模式，不需使用者互動。與 /qn 相同
/passive 自動安裝模式 - 只有進度列。與 /qb 相同


記錄選項：
/l[i|w|e|a|r|u|c|m|o|p|v|x|*] 將與安裝相關的一般訊息記錄為記錄檔。如果您未指名路徑和檔案名稱，系統會將記錄檔建立在目前使用者的暫時資料夾中。

i - 狀態訊息
w - 非嚴重警告
e - 所有錯誤訊息
a - 動作起始時
r - 特定動作記錄
u - 使用者要求
c - 初始 UI 參數
m - 記憶體用盡或嚴重結束資訊
o - 磁碟空間用盡訊息
p - 終端機內容
v - 詳細資訊輸出
x - 額外偵錯資訊
* - 記錄所有資訊，除了 v 和 x 選項

預設值為 /lvoicewarmup

Setup.exe /l"C:\Install.log"


修復選項：
/f[p|e|c|m|s|o|d|a|u] 修復產品。
p - 只有在檔案遺失
o - 如果檔案遺失或已安裝了較舊的版本 (預設值)
e - 如果檔案遺失或已安裝了相同或較舊的版本
d - 如果檔案遺失或已安裝了不同的版本
c - 如果檔案遺失或總合檢查碼跟計算的數值不相符
a - 強制重新安裝所有檔案
u - 所有需要的使用者特定的登錄項目 (預設值)
m - 所有需要的電腦特定的登錄項目 (預設值)
s - 所有現存的捷徑 (預設值)


預設值為 /foums

Setup.exe /f
