Git & Github

Git adalah sistem kontrol versi (VCS *Version control system) terdistribusi yang memungkinkan Dev melacak perubahan kode dan kolaborasi dengan lebih efisien.

3 Fitur Git adalah Branching (Cabang), Commits (catatan perubahan), Staging area (Area persiapan) Untuk commits cara bacanya yang merah yang lama sedangkan yang hijau adalah yang baru

Github adalah blablabla

3 fitur github adalah pull request, repository, action: automatic testing (pengujian otomatis)

Kesimpulan

Git adalah seoerti buku catatan itu di rak buku yang disimpan di rumah untuk mencatat semua ide dan perubahan dalam proyek pribadi mu. Kamu bisa menulis dan mengedit catatan tersebut kapan saja tanpa membagikan ke orang lain.

Github adalah menyimpan buku catatan itu di rak buku perpustakaan yang bisa diakses teman-temanmu. selain menyimpan catatn, perpustakaan juga memungkinkan teman-teman untuk melihat, memberikan masukan dan berkolaborasi denganmu dalam proyek tersebut
