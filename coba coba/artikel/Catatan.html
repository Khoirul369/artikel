Web site terdiri dari 3 yakni HTTP HTML URI

Hypertext transfer protocol Didalam Http ada respon dan request dan pada respon ada yang namanya status code diantara nya.

2xx = Kalo 2xx itu berhasil, Bila responnya 201 berarti respon succses.
4xx = Kalo 4xx itu kesalahan user, Bila responnya 403 berarti ditolak, Bila 401 authotization kurang.
5xx = Kalo 5xx itu kesalahan server, Bila responnya 503 berarti server tidak tersedia.
Method ada 4= Get, Post, Put, Delete
Html= Hypertext Markup Language URI= Uniform Resource Identifer
