Este é um guia de referência rápida para as linhas de comandos comuns usadas com o Setup.exe.

Setup.exe /Opção <Parâmetro obrigatório> [Parâmetro opcional]

Opções de instalação:
/a Sinaliza que você quer criar uma imagem de servidor. Instalação administrativa - instala um produto na rede

Setup.exe /a
/a /q(n) O botão /q(n) nas linhas de comando não tem mais suporte. Consulte a versão mais atualizada do Guia de Implantação do CorelDRAW Graphics Suite para ver os botões suportados.

Setup.exe /a /q(n)
/s Usado para empurrar PROFILEID para as Estações de Trabalho. A propriedade pública PROFILEID é usada para logar todos os usuários na conta ao empurrar as credenciais da conta para as estações de trabalho.

Setup.exe /s PROFILEID=XXXXXXXXXXXXXXX
/x
/uninstall		 Sinaliza que você quer desinstalar o software. Instalação administrativa - instala um produto na rede

Setup.exe /x


Opções de exibição:
/q[n|b|r|f] Define a extensão na qual a interface do usuário da instalação será exibida.
n - sem UI
b - IU básica
r - IU reduzida
f - IU completa (padrão)

Setup.exe /q
/quiet Modo silencioso, sem interação do usuário. O mesmo que /qn
/passive Modo sem supervisão - somente barra de andamento. O mesmo que /qb


Opções de registro:
/l[i|w|e|a|r|u|c|m|o|p|v|x|*] Registra informações gerais sobre a instalação em um arquivo de registro. Se você não especificar um caminho e nome de arquivo, o arquivo de registro será criado na pasta temporária do usuário atual.

i - mensagens de status
w - avisos não fatais
e - todas as mensagens de erro
a - inicialização de ações
r - registros específicos de ações
u - solicitações de usuário
c - parâmetros iniciais de IU
m - informações sobre saída falta ou falta de memória
o - mensagens sobre falta de espaço em disco
p - propriedades de terminal
v - saída detalhada
x - informações extras sobre depuração
* - registro de todas as informações, exceto para as opções v e x

O padrão é /lvoicewarmup

Setup.exe /l"C:\Install.log"


Opções de reparo:
/f[p|e|c|m|s|o|d|a|u] Repara um produto.
p - somente se o arquivo estiver faltando
o - se o arquivo estiver faltando ou uma versão mais antiga estiver instalada (padrão)
e - se o arquivo estiver faltando ou uma versão igual ou mais antiga estiver instalada
d - se o arquivo estiver faltando ou uma versão diferente estiver instalada
c - se o arquivo estiver faltando ou uma soma de verificação não corresponder ao valor calculado
a - força todos os arquivos a serem reinstalados
u - todas as entradas do registro específicas por usuário necessárias (padrão)
m - todas as entradas do registro específicas por computador necessárias (padrão)
s - todos os atalhos existentes (padrão)


O padrão é /foums

Setup.exe /f


Configuração de propriedades públicas:

[PROPERTY=PropertyValue]
