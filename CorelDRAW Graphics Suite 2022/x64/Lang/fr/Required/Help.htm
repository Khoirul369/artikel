Ceci est un guide de référence rapide relatif aux lignes de commande courantes utilisées avec Setup.exe.

Setup.exe /Option <Paramètre obligatoire> [Paramètre facultatif]

Options d’installation :
/a Indique que vous souhaitez créer une image serveur. Installation administrative - Installe un produit sur le réseau

Setup.exe /a
/a /q(n) Le commutateur /q(n) dans les lignes de commandes n'est plus pris en charge. Veuillez consulter la dernière version du guide de déploiement de CorelDRAW Graphics Suite pour savoir quels sont les commutateurs pris en charge.

Setup.exe /a /q(n)
/s Permet une transmission de type Push de PROFILEID vers les stations de travail. La propriété publique PROFILEID permet de connecter tous les utilisateurs au compte grâce à une transmission de type Push des informations d’identification du compte vers les stations de travail.

Setup.exe /s PROFILEID=XXXXXXXXXXXXXXX
/x
/uninstall		 Indique que vous souhaitez désinstaller le logiciel. Installation administrative - Installe un produit sur le réseau

Setup.exe /x


Options d’affichage :
/q[n|b|r|f] Définit l’étendue de l’affichage de l’interface utilisateur lors de l’installation.
n - Pas d’interface utilisateur
b - Interface utilisateur de base
r - Interface utilisateur réduite
f - Interface utilisateur complète (option par défaut)

Setup.exe /q
/quiet Mode silencieux, aucune interaction avec l’utilisateur. Identique à /qn
/passive Mode autonome - barre de progression uniquement. Identique à /qb


Options de journalisation :
/l[i|w|e|a|r|u|c|m|o|p|v|x|*] Consigne les informations générales concernant l’installation dans un fichier journal. Si vous n’indiquez pas de chemin ni de nom de fichier, le fichier journal est créé dans le dossier temporaire de l’utilisateur actuel.

i - Messages d’état
w - Avertissements récupérables
e - Tous les messages d’erreur
a - Démarrage des actions
r - Enregistrements spécifiques à certaines actions
u - Requêtes de l’utilisateur
c - Paramètres d’origine de l’interface utilisateur
m - Mémoire insuffisante ou informations de sortie irrécupérables
o - Messages d’espace disque insuffisant
p - Propriétés du terminal
v - Mode documenté
x - Autres informations de débogage
* - Enregistrer toutes les informations, sauf pour les options v et x

Le paramètre par défaut est /lvoicewarmup

Setup.exe /l"C:\Install.log"


Options de réparation :
/f[p|e|c|m|s|o|d|a|u] Répare un produit.
p - uniquement si un fichier est manquant
o - si un fichier est manquant ou si une version antérieure est installée (option par défaut)
e - si un fichier est manquant ou si une version équivalente ou antérieure est installée
d - si un fichier est manquant ou si une version différente est installée
c - si un fichier est manquant ou si la somme de contrôle ne correspond pas à la valeur calculée
a - force la réinstallation de tous les fichiers
u - toutes les entrées de registre requises spécifiques à l’utilisateur (option par défaut)
m - toutes les entrées de registre requises spécifiques à l’ordinateur (option par défaut)
s - tous les raccourcis existants (option par défaut)


L’option par défaut est /foums

Setup.exe /f


Définition des propriétés publiques :

[PROPERTY=PropertyValue]
