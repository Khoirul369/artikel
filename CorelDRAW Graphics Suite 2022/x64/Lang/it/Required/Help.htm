Questa è una guida di consultazione rapida per le righe di comando comuni utilizzate con Setup.exe.

Setup.exe /Opzione<Parametro obbligatorio> [Parametro opzionale]

Opzioni di installazione:
/a Segnala che l'utente desidera creare un'immagine per il server. Installazione amministrativa - Consente di installare un prodotto sulla rete

Setup.exe /a
/a /q(n) Il parametro /q(n) nelle righe di comando non è più supportato. Per informazioni sui parametri supportati, consulta l'ultima versione della Guida alla distribuzione di CorelDRAW Graphics Suite.

Setup.exe /a /q(n)
/s Utilizzato per integrare PROFILEID nelle postazioni di lavoro. La proprietà pubblica di PROFILEID viene utilizzata per registrare tutti gli utenti all'account integrando le credenziali dell'account nelle postazioni di lavoro.

Setup.exe /s PROFILEID=XXXXXXXXXXXXXXX
/x
/uninstall		 Segnala che l'utente desidera disinstallare il software. Installazione amministrativa - Consente di installare un prodotto sulla rete

Setup.exe /x


Opzioni di visualizzazione:
/q[n|b|r|f] Consente di impostare la misura con cui viene visualizzata l'interfaccia utente dell'installazione.
n - Nessuna IU
b - IU di base
r - IU ridotta
f - IU intera (predefinita)

Setup.exe /q
/quiet Modalità Quiet, nessuna interazione dell'utente. Uguale a /qn
/passive Modalità Unattended - solo barra di avanzamento. Uguale a /qb


Opzioni di accesso:
/l[i|w|e|a|r|u|c|m|o|p|v|x|*] Registra le informazioni generali sull'installazione in un file di registro. Se non viene specificato un percorso o un nome del file, il file di registro viene creato nella cartella temporanea dell'utente corrente.

i - Messaggi di stato
w - Avvertenze non gravi
e - Tutti i messaggi di errore
a - Avvio di azioni
r - Registrazioni specifiche di azioni
u - Richieste dell'utente
c - Parametri iniziali dell'IU
m - Informazioni di memoria esaurita o di abbandono della procedura
o - Messaggi di spazio su disco insufficiente
p - Proprietà del terminale
v - Output prolisso
x - Informazioni supplementari di debugging
* - Registra tutte le informazioni ad eccezione delle opzioni v e x

L'impostazione predefinita è /lvoicewarmup

Setup.exe /l"C:\Install.log"


Opzioni di riparazione:
/f[p|e|c|m|s|o|d|a|u] Consente di riparare un prodotto.
p - solo se manca il file
o - se manca il file o è installata una versione precedente (predefinita)
e - se manca il file o è installata una versione equivalente o precedente
d - se manca il file o è installata una versione diversa
c - se manca il file o il checksum non corrisponde al valore calcolato
a - forza la reinstallazione di tutti i file
u - tutte le immissioni obbligatorie nel registro specifiche dell'utente (predefinita)
m - tutte le immissioni obbligatorie nel registro specifiche del computer (predefinita)
s - tutti i tasti di scelta rapida esistenti (predefinita)


L'impostazione predefinita è /foums

Setup.exe /f


Impostazione delle proprietà pubbliche:

[PROPERTY=PropertyValue]
