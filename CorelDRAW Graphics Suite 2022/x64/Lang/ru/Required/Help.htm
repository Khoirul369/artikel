Это краткое справочное руководство по общим командным строкам, используемым для программы Setup.exe.

Setup.exe /параметр <обязательный параметр> [необязательный параметр]

Параметры установки:
/a Указывает на необходимость создания образа сервера. Административная установка: установка ПО в сети

Setup.exe /a
/a /q(n) Параметр командной строки /q(n) больше не поддерживается. Поддерживаемые параметры командной строки см. в последней версии руководства по развертыванию CorelDRAW Graphics Suite.

Setup.exe /a /q(n)
/s Используется для активной отправки PROFILEID на рабочие станции. Открытое свойство PROFILEID используется для входа всех пользователей в учетную запись путем активной отправки учетных данных на рабочие станции (без запроса).

Setup.exe /s PROFILEID=XXXXXXXXXXXXXXX
/x
/uninstall		 Указывает на необходимость удаления программного обеспечения. Административная установка: установка ПО в сети

Setup.exe /x


Параметры отображения:
/q[n|b|r|f] Задает степень детализации отображения интерфейса пользователя во время установки.
n - без интерфейса
b - основной интерфейс
r - сокращенный интерфейс
f - полный интерфейс (по умолчанию)

Setup.exe /q
/quiet Тихий режим (без отображения пользовательского интерфейса). Аналогичен параметру /qn
/passive Автоматический режим (отображается только индикатор процесса установки). Аналогичен параметру /qb


Параметры ведения журнала:
/l[i|w|e|a|r|u|c|m|o|p|v|x|*] В журнале сохраняются общие сведения об установке. Если путь и имя файла не заданы, файл журнала создается во временной папке текущего пользователя.

i - сообщения о состоянии
w - некритические предупреждения
e - все сообщения об ошибках
a - запуск действий
r - записи, относящиеся к конкретным действиям
u - запросы пользователя
c - начальные параметры интерфейса пользователя
m - информация о нехватке памяти или аварийном завершении
o - сообщения о нехватке места на диске
p - свойства терминала
v - вывод подробных сведений
x - дополнительная отладочная информация
* - заносить в журнал все сведения, кроме параметров v и x

По умолчанию используется параметр /lvoicewarmup

Setup.exe /l"C:\Install.log"


Параметры восстановления:
/f[p|e|c|m|s|o|d|a|u] Восстанавливает программное обеспечение.
p - только если файл отсутствует
o - если файл отсутствует или имеется файл более ранней версии (по умолчанию)
e - если файл отсутствует или имеется файл с таким же или предыдущим номером версии
d - если файл отсутствует или имеется файл с несоответствующим номером версии
c - если файл отсутствует или контрольная сумма не совпадает с подсчитанным значением
a - выполняется принудительная переустановка всех файлов
u - все необходимые записи реестра для пользователя (по умолчанию)
m - все необходимые записи реестра для компьютера (по умолчанию)
s - все существующие ярлыки (по умолчанию)


По умолчанию используется параметр /foums

Setup.exe /f


Настройка общих свойств:

[PROPERTY=PropertyValue]
